# Pizza Sales Analysis Dashboard

---

## **Project Overview**
- **Goal:** To transform raw pizza sales data into actionable business insights.
-  **Tools Used:** **Power-BI**, **DAX**, **POWER QUERY:**.
-  **Dataset:** 20,000+ transactional records analyzing sales, quantity, and categories.
---
## **Key Business Insights**
- **Revenue Drivers:**
-  **Large (L)** size pizzas are the main engine, contributing **46.68** of total revenue.
-  **Strategy:** Focus on family-sized meal deals to maintain this high-margin segment.

- **Product Performance:**
- **The Pepperoni Pizza** is the #1 best-seller by volume.
- **Top 3 revenue-generating pizzas: **Pepperoni**, ** Thai Chicken**, and **Barbecue Chicken**.

## **Sales Seasonality:**
- Peak performance occurs during **July and August**.
- Significant sales dip in **April and May**.
- **Recommendation:** Run spring-spesific promotions to boost off-peak sales.

- **Efficiency Metrics:**
- **Average Order Value (AOV):** $37.83.
- **Target:** Increase AOV to **$40** through dessert or slide-dish upselling.

---

## **Technical Features**

- **Data Engineer:**
- Cleaned and prepared data using **Power Query**.
- Extracted **Hour** and **Month** for time-series and peak-hour analysis.

- **Analytical Modeling:**
- Built dynamic **DAX Measures** for Total Revenue, Total Orders, and Average Order and Value .
- Applied **Top N Filters** to identify high-performing products.
- **Modern Workflow:** Leveraged **DAX Query View** and **TDML** for advanced metadata management and model optimization.

----
## **How to use** 
1.Download the `pbix` file from this repository.
2.Open it using **Power BI Deskop**.
3.Use the filters on the right to interact with the data and explore different pizza categories.
